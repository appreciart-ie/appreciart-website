# Appreciart IE — Design System & Project Reference

## Project

- **Name:** Appreciart IE
- **Description:** Public website for a private tattoo studio and gallery in Dublin
- **Repo:** https://github.com/appreciart-ie/appreciart-website.git
- **Stack:** Vanilla JS, HTML, CSS — no frameworks, no build tools
- **Host:** Vercel (static + serverless API routes later)

## Folder Structure

```
/css
/js
/images/logos
/images/artists
/images/studio
/images/dates
```

## Typography

- **Font family:** Poppins (Google Fonts)
- **Weights used:**
  - 300 — Light (body, secondary text)
  - 400 — Regular (body, general content)
  - 700 — Bold (titles, headings)
  - 900 — Black (hero titles, strong emphasis)
- **Hierarchy:** Strong contrast between weights — titles in 700/900, body in 300/400

## Colours

| Name            | Hex       | Usage                        |
|-----------------|-----------|------------------------------|
| Black           | `#000000` | Primary text, headings       |
| White           | `#FFFFFF` | Backgrounds                  |
| Main Grey       | `#636363` | Secondary text, UI elements  |
| Secondary Grey  | `#9a9a9a` | Tertiary text, subtle detail |

## Tone & Visual Direction

- Editorial, gallery, magazine aesthetic
- Clean white background
- High contrast black type
- Minimal colour use
